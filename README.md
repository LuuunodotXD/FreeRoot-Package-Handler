esse é um sistema e gerenciador de pacotes que eu fiz para o FreeRoot Linux.
funções:
instalar packages com scripts shell;
instalar binários em C compilados;
resolver dependências automaticamente;
verificar checksum ao baixar packages (sha-256);
registrar o que está instalado;
excluir pacotes (incluindo os arquivos).

-- this is a package handler that i made for the FreeRoot Linux.
functions:

automatically resolve dependencies;
check checksum when downloading packages (sha-256)
register which packages are installed;
delete packages (including files).
